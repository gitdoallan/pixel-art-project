Como utilizar esta página?


Esta página contém um script que carrega a cor preta e mais 3 cores aleatórias.

Clique em uma das cores para poder pintar o quadro de pixels localizado na parte inferior.

Você pode trocar as cores aleatóriamente clicando no botão "Trocar cores".

A opção "Cores Xablau" irá alterar as cores aleatóriamente a cada meio segundo.

O botão "Limpar" vai resetar as cores do quadro de pixels.

Em "Tamanho" você pode redefinir o tamanho do quadro e utilizar sua criatividade para desenhar com a nova quantidade pixels gerada :D
